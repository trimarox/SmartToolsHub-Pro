# SmartToolsHub

SmartToolsHub is a professional all-in-one digital tools platform. It provides a collection of free, AI-powered utilities accessible directly in the browser. No sign-up required — start boosting your productivity instantly!

## Features

- **AI Paraphraser:** Rewrite and enhance text.
- **YouTube Title Generator:** Generate viral, click-worthy video titles.
- **TikTok Hook Generator:** Create hooks that stop the scroll in seconds.
- **QR Code Generator:** Generate QR codes for URLs, text, WiFi, and more.
- **Password Generator:** Create strong and secure passwords.
- **Background Remover:** Remove image backgrounds instantly (placeholder ready for AI integration).
- **Resume Builder:** Build professional resumes with customizable input.

## Technologies

- HTML5, CSS3, JavaScript
- Tailwind CSS for responsive and modern design
- Font Awesome for icons
- QRCode.js for QR code generation
